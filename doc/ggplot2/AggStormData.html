AggStormDataR Documentation

Aggregated US Storm Data

Description

A dataset containing aggregated US Weather storm Data from 1993 to 2011. The variables are as follows:

Usage

data(AggStormData)

Format

A data frame with 17,000 rows and 8 variables

"YEAR","STATE","EVTYPE","COUNT","FATALITIES","INJURIES","PROPDMG","CROPDMG"

Details

  • YEAR

  • STATE

  • EVTYPE: EVENT TYPE

  • COUNT

  • FATALITIES

  • INJURIES

  • PROPDMG: PROPERTY DAMAGE

  • CROPDMG: CROP DAMAGE
